# Installing Rust
## Linux😎
Very easy, run the following command:
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
## Windows💩
The Rust installer is available on [Rustup.rs](https://rustup.rs/).
